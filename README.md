# FWTI — 恋爱废物人格测试

**Fèiwù Type Indicator** — 自嘲式恋爱人格测试，测测你在恋爱里是哪种废物。

> 🔗 在线体验：[fwti.innei.dev](https://fwti.innei.dev)

## 关于

30 道主线题 + 3 道隐藏彩蛋 + 10 个叠加隐藏标签 + 8 种隐藏人格，四大维度交叉分析，精准匹配你的恋爱废物类型。

仅供娱乐，请勿用于相亲、挽回、分手、发律师函或自我攻击。

## 四大维度

| 维度 | 极性 A | 极性 B | 核心问题 |
| --- | --- | --- | --- |
| **主动性** | G（Go·冲） | D（Dwell·蹲） | 冲锋还是蹲坑？ |
| **情绪表达** | Z（Zha·炸） | R（Ren·忍） | 炸了还是忍了？ |
| **亲密需求** | N（Nian·黏） | L（Li·离） | 黏人精还是独处王？ |
| **安全感** | Y（Yi·疑） | F（Fo·佛） | 疑心病还是佛祖？ |

四字母组合 → 2⁴ = **16 种恋爱废物人格 + 8 种隐藏人格**。

## 技术栈

- **Solid.js** — 响应式 UI 框架
- **Vike** — SSR 框架
- **Vite** — 构建工具
- **html-to-image** — 结果分享图生成
- 部署于 **Vercel**

## 本地开发

```bash
pnpm install
pnpm dev
```

构建：

```bash
pnpm build
```

## 学术依据

四大维度设计参考成人依恋理论（Hazan & Shaver, 1987; Brennan, Clark & Shaver, 1998）、情绪调节量表（Gross & John, 2003 ERQ）、接近–回避动机（Carver & White, 1994 BIS/BAS; Gable, 2006）。

但本测试 **未经心理测量学信效度验证**，不是诊断工具。认真想了解自己的依恋模式，请做正规 ECR-R 量表。
